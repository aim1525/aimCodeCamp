# Code Camp Starter Kit (React + TypeScript)

A fast, fun portfolio starter for a 15-minute live micro-lab. Customize two files, unlock the secret skin, and submit your repo link.

## Prereqs
- Node.js 18+ (install from the official Node.js website)
- Git

## Setup (3 minutes)
1. Click **Use this template** on GitHub to create your copy.
2. Clone your new repo.
3. Run `npm install`.
4. Run `npm run dev` and open the local URL.

## 5-Minute Challenges
1. **Challenge 1 (Profile):**
   - Open `src/data/profile.ts`
   - Change `name` and `tagline`
2. **Challenge 2 (Projects):**
   - Open `src/data/projectsData.ts`
   - Add **one** new project object to `projectsData`
3. **Optional:** Find the easter egg to unlock hidden themes + confetti.

## Submission
1. Commit and push your changes to GitHub.
2. Submit your repo link at: **[PLACEHOLDER_SUBMISSION_URL]**

## Troubleshooting
- **Node.js not installed?** Install Node.js from the official Node.js website, then try again.
- **No local setup?** Use the GitHub web editor to update `profile.ts` and `projectsData.ts`, commit the changes, and submit your repo link anyway.
